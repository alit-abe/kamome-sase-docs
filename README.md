# Kamome SASE ドキュメント

Kamome SASE製品の公式ドキュメントリポジトリです。

## 概要

このリポジトリには、Kamome SASE（高度なネットワークセキュリティソリューション）の各コンポーネントに関するマニュアルが含まれています。

## ドキュメント構成

- **[エージェント](docs/agent/)** - クライアントPC用エージェントのマニュアル
- **コネクター** - リモートネットワーク用コネクターのマニュアル（準備中）
- **システム管理** - システム管理者向けマニュアル（準備中）
- **テナント管理** - テナント管理者向けマニュアル（準備中）
- **認証サーバー** - SSO認証サーバーのマニュアル（準備中）
