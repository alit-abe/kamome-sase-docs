# Kamome SASE マニュアル

## 製品概要

Kamome SASEは、高度なネットワークセキュリティを実現するための統合セキュリティソリューションです。

## コンポーネント

### ピア（Peer）
- **[エージェント](agent/index.md)**: クライアントPC側のエージェント
- **[コネクター](connector/index.md)**: リモートネットワーク側のコネクター

### センターサーバー（Center Server）
- **[システム管理](system-admin/index.md)**: 集中管理システム
- **[テナント管理](tenant-admin/index.md)**: テナント管理システム

### 認証サーバー（Auth Server）
- **[認証サーバー](auth-server/index.md)**: SSO（シングルサインオン）機能

## はじめに

Kamome SASEを初めて使用する場合は、以下の順序でマニュアルをご覧ください：

1. システム要件の確認
2. 認証サーバーのセットアップ
3. センターサーバーのセットアップ
4. エージェント/コネクターのインストール
5. 各管理画面での設定
