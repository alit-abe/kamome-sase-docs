# エージェント マニュアル

## 概要

エージェントは、クライアントPC側にインストールするKamome SASEのコンポーネントです。
クライアントPCからセキュアなネットワークアクセスを提供します。

## 主な機能

- セキュアなネットワーク接続
- アクセスポリシーの適用
- トラフィック監視とログ記録
- 自動接続管理

## 対応OS

- Windows 10/11（64bit）

## システム要件

### 最小要件
- CPU: 2コア以上
- メモリ: 4GB以上
- ディスク空き容量: 500MB以上
- ネットワーク: インターネット接続必須

### 推奨要件
- CPU: 4コア以上
- メモリ: 8GB以上
- ディスク空き容量: 1GB以上
- ネットワーク: 高速インターネット接続
